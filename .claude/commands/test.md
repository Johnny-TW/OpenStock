執行測試指令：

## 後端單元測試
```bash
cd backend && pnpm test
```

## 後端測試覆蓋率
```bash
cd backend && pnpm test:cov
```

## 後端 E2E 測試
```bash
cd backend && pnpm test:e2e
```

## 後端 Watch 模式
```bash
cd backend && pnpm test:watch
```

## 前端 Lint 檢查
```bash
cd frontend && pnpm lint
```

## 全部 Lint（透過 lint-staged）
```bash
pnpm lint-staged
```
